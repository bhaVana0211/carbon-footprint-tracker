# 🌍 EcoTrace: Interactive Carbon Footprint Tracker

An AI-assisted, single-page web application designed to help individuals understand, track, and reduce their daily environmental footprint through real-time feedback and behavioral suggestions. 

Built during the **#PromptWarsVirtual** challenge using **Google Antigravity IDE**.

## 🌱 Core Features
* **Multi-Category Footprint Assessment**: Inputs data across transit (flights, daily commute), dietary choices (meat consumption), and domestic utility consumption.
* **Instant Dynamic Calculations**: Recalculates metrics instantly upon user updates using optimized clamping metrics to prevent negative or boundary overflows.
* **Actionable Smart Eco-Swaps**: Replaces complex climate metrics with understandable, real-life product swaps and eco-friendly choices.
* **Comprehensive Test Suite**: Automated verification pipeline built with Jest to validate calculation accuracy.

## 🛠️ Technology Stack
* **Frontend**: Semantic HTML5, CSS3 Grid/Flexbox layouts, Vanilla ES6+ JavaScript.
* **Development Environment**: Google Antigravity IDE (driven by Gemini 3.5 capabilities for multi-file generation and agentic edge-case code verification).
* **Testing Pipeline**: Jest testing framework for script logical checks.
* **Deployment Platform**: GitHub Pages.
